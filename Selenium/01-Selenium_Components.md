# Selenium Components

## What is Selenium?

Selenium is an open-source automation testing tool used for automating web applications.

The Selenium Suite consists of four major components:

1. Selenium IDE
2. Selenium WebDriver
3. Selenium Grid
4. Selenium RC (Deprecated)

---

# 1. Selenium IDE

## What is Selenium IDE?

Selenium IDE is a record-and-playback tool used to create automation scripts without writing code.

### Features

* Record user actions
* Playback recorded scripts
* Easy to learn
* Browser extension

### Usage

Best suited for beginners and quick test creation.

### Limitation

Not suitable for complex automation frameworks.

---

# 2. Selenium WebDriver

## What is Selenium WebDriver?

Selenium WebDriver is the core component of Selenium used for browser automation.

It directly interacts with browsers and performs actions like:

* Click
* Type
* Select
* Navigate

### Features

* Supports multiple browsers
* Supports multiple programming languages
* Faster than Selenium RC

### Usage

Used for developing automation frameworks and real-world automation projects.

---

# 3. Selenium Grid

## What is Selenium Grid?

Selenium Grid allows execution of test cases on multiple browsers and machines simultaneously.

### Features

* Parallel execution
* Cross-browser testing
* Cross-platform testing

### Benefits

* Reduces execution time
* Supports distributed execution

### Example

Run the same test simultaneously on:

* Chrome
* Firefox
* Edge

---

# 4. Selenium RC (Remote Control)

## What is Selenium RC?

Selenium RC was the older Selenium automation tool before WebDriver.

### Features

* Required Selenium Server
* Supported multiple browsers

### Limitations

* Slow execution
* Complex architecture

### Current Status

Deprecated and replaced by Selenium WebDriver.

# Interview Answer

**Selenium consists of four components: Selenium IDE, Selenium WebDriver, Selenium Grid, and Selenium RC. Selenium IDE is used for record and playback, WebDriver is used for browser automation, Grid is used for parallel execution across multiple browsers and machines, and Selenium RC is the deprecated predecessor of WebDriver.**
